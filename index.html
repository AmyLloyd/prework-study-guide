HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Tags in the head include title, style, base, link, meta and script.
  • Tags common in the body include header, img, main, section.
  • Main is the parent element that contains the child element section.

CSS

  • CSS is in charge of the styling of a webpage.
  • This includes colours, aesthetics, visual layout and organisation.
  • The most common way to style a webpage is through an external CSS style sheet.
  • This is preferred as giving each section of code its own file is an example of the separation of concerns. This links to the concept that each section of code has its own responsibility.
  • There are punctuation characters that have significant functions and meanings in CSS.

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Variable and functions are similar as they must be declared.
  • Functions are then called in order to execute.
